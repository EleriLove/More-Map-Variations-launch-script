# this script will work on any linux system.... probably... #

to install this script you need to pick a directory you want the mod isntalled in
the directory needs to be completely empty. 

in order to do this you can use the mkdir command in terminal, or in your gui
just... make a folder like normal, 

take note of the location of this folder you will need it later.

when you're ready with the location you want

use ```git clone https://github.com/EleriLove/More-Map-Variations-launch-script```
anywhere in the terminal, this will download all scripts required for installing the mod.

once you have that downloaded use the ``cd More-Map-Variations-launch-script`` command in order to go into the install directory

once you're in there simply type ``./general_mmv_installer.sh <path to install directory>`` 
remember to replace ``<path to install directory>`` with the location of your folder you created earlier.
if everything worked you should be able to move to the install location in gui and see a mod folder along with a launch_nightreign.sh file, 

if you do, then the script worked! just double lick the launch file and nightreign should boot up. you can add this file to steam if you want to launch via steam like any non steam game.
